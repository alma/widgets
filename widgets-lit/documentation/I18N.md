# Internationalization (i18n)

The Lit widget uses a lightweight i18n system defined in `src/i18n/messages.ts`.

---

## Supported Locales

- `en`, `fr`, `de`, `es`, `it`, `pt`, `nl`
- Regional variants are also supported (for example `fr-FR`, `en-US`, `nl-BE`).

---

## How It Works

All strings live in a single `messages` object:

```ts
export const messages = {
  en: {
    'paymentPlans.loading': 'Loading...',
    'modal.titleNormal': 'Pay in installments by credit card with Alma.',
  },
  fr: {
    'paymentPlans.loading': 'Chargement...',
    'modal.titleNormal': 'Payez en plusieurs fois par carte bancaire avec Alma.',
  },
}
```

Use the `t()` helper in components:

```ts
import { t } from '../i18n'

const lang = (this.locale.split('-')[0] as any) || 'fr'
const label = t(lang, 'paymentPlans.loading')
```

---

## Locale‑Specific Formatting

- Deferred day notation uses `J+15` for French and `D+15` for other locales.
- Date formats are driven by translation keys (see `date.format`).

---

## Adding a Locale

1. Add a locale entry in `src/i18n/messages.ts`.
2. Add the locale code in `src/constants.ts` (`AVAILABLE_LOCALES`).

---

## Adding a New Translation Key

1. Add the key to all locales in `messages.ts`.
2. Use it via `t(lang, 'your.key')`.
